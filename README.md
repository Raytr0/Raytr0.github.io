## Tools Used 🛠️
* [<b>GitHub Pages</b>](https://create-react-app.dev/docs/deployment/#github-pages) - To host my static website (HTML, CSS, JS).
* [<b>Modern CSS</b>](https://developer.mozilla.org/en-US/docs/Web/CSS) - Custom CSS with CSS variables, gradients, and animations for a sleek design.
* [<b>Typed.js</b>](https://mattboldt.com/demos/typed-js/) - JavaScript Library for dynamic typing animations.
* [<b>Google Fonts</b>](https://fonts.google.com/) - Professional typography with Inter font family.
